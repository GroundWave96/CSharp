# CSharp

### Tipos

 Todas as variáveis e constantes têm um tipo.
 
 Nome da variavel pode conter: a-z | A-Z | 0-9(não no início) | _
 
 Não pode conter símbolos (@ $ .)
```
var :       escolhe a variável automaticamente
object :    
dynamic :   
sbyte :    -128 a 127                                                      (8 bits ou 1 bytes)
byte :      0 a 255                                                        (8 bits ou 1 bytes)
short :    -32.768 a 32.767                                                (16 bits ou 2 bytes)
ushort :    0 a 65.535                                                     (16 bits ou 2 bytes)
int :      -2.147.483.648 a 2.147.483.647                                  (32 bits ou 4 bytes)
uint :      0 a 4.294.967.295                                              (32 bits ou 4 bytes)
long :     -9.223.372.036.854.775.808 a 9.223.372.036.854.775.807          (64 bits ou 8 bytes)
ulong :     0 a 18.446.744.073.709.551.615                                 (64 bits ou 8 bytes)
float :    ±1,5 x 10^−45 a ±3,4 x 10^38 precisão ~6 a 9 dígitos            (32 bits ou 4 bytes)
double :   ±5,0 × 10^−324 a ±1,7 × 10^308 precisão ~15 a 17 dígitos        (64 bits ou 8 bytes)
decimal :  ±1,0 x 10^-28 a ±7,9228 x 10^28 precisão ~28 a 29 dígitos       (128 bits ou 16 bytes)
bool :      true ou false | padrão false                                   (1 bit)
char :      caracteres | 'a'                                               (16 bits ou 2 bytes)
string :    texto | "Hello World!"                                         (16 bits ou 2 bytes por caracter)
```
